# My_fourth_Web_project
A responsive personal portfolio built using HTML, CSS, and JavaScript featuring a To-Do App with LocalStorage and a Product Listing page with filtering and sorting functionality.

# 🌐 Personal Portfolio Website

This repository contains a **responsive personal portfolio website** built using **HTML, CSS, and JavaScript**.  
It showcases multiple frontend projects and demonstrates core web development concepts through real, working applications.


## 🚀 Projects Included

### 1️⃣ Personal Portfolio Website
- Multi-section layout: **About, Projects, Contact**
- Fully responsive design
- Clean and beginner-friendly UI

### 2️⃣ To-Do List Application
- Add and delete tasks
- Tasks are saved using **LocalStorage**
- Data persists even after page refresh

### 3️⃣ Product Listing Page
- Dynamic product display
- Filter products by **category** and **price**
- Sort products by **price** or **rating**
- Implemented using JavaScript array methods and DOM manipulation


## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & responsiveness
- **JavaScript (ES6)** – Logic & interactivity
- **LocalStorage API** – Data persistence
